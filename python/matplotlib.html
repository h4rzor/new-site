Getting started with Matplotlib


Overview of Matplotlib


Matplotlib is a low level plotting library.It works well with Numpy arrays as well as Python lists.


You can install Matplotlib by simply typing in the terminal or CMD this:

pip install matplotlib

Next thing is using it. Simply type this:

import matplotlib.pyplot as plt

I'm going to be using a subset of matplotlib library called pyplot for these examples.



Drawing a line:


import numpy as np import matplotlib.pyplot as plt fig, ax = plt.subplots() x = np.array([0,6]) y = np.array([0,50]) plt.plot(x,y) fig



Plotting without lines:

import numpy as np import matplotlib.pyplot as plt fig, ax = plt.subplots() x = np.array([0,6]) y = np.array([0,50]) plt.plot(x,y, 'o') fig


Plotting linestyles:

import matplotlib.pyplot as plt import numpy as np fig, ax = plt.subplots() ypoints = np.array([3, 8, 1, 10]) plt.plot(ypoints, linestyle = 'dotted') fig
import numpy as np import matplotlib.pyplot as plt models=['00','01', '02', '03', '04', '05', '06', '07', '08', '09', '10',\ '11', '12', '13', '14', '15', '16'] fig = plt.figure() ax = fig.add_subplot(111) x = np.linspace(-1,1,100) y = np.sin(x) clrs_list=['k','b','g','r'] # list of basic colors styl_list=['-','--','-.',':'] # list of basic linestyles for i in range(0,16): clrr=clrs_list[i // 4] styl=styl_list[i % 4] modl=models[i+1] frac=(i+1)/10.0 ax.plot(x,y+frac,label=modl,color=clrr,ls=styl) plt.legend() fig

As you can see, you can iterate through different linestyles and colors.




Scattering:

import matplotlib.pyplot as plt import numpy as np x = np.array([5,7,8,7,2,17,2,9,4,11,12,9,6]) y = np.array([99,86,87,88,111,86,103,87,94,78,77,85,86]) fig, ax = plt.subplots() plt.scatter(x, y) # Both arrays have to be equal size and it combines the two points into its intersection fig

Different colors:

import matplotlib.pyplot as plt import numpy as np x = np.array([5,7,8,7,2,17,2,9,4,11,12,9,6]) y = np.array([99,86,87,88,111,86,103,87,94,78,77,85,86]) colors = np.array(["red","green","blue","yellow","pink","black","orange","purple","beige","brown","gray","cyan","magenta"]) fig, ax = plt.subplots() plt.scatter(x, y, c=colors) # Example of different colors fig


Bars:

import matplotlib.pyplot as plt import numpy as np x = np.array(["A", "B", "C", "D"]) y = np.array([3, 8, 1, 10]) fig, _ = plt.subplots() plt.bar(x,y) # Perfect for quick look at the distribution of certain things fig


Hisograms:

import matplotlib.pyplot as plt import numpy as np x = np.random.normal(170, 10, 250) fig, _ = plt.subplots() plt.hist(x) # Distribution chart example fig


Pie chart:

import matplotlib.pyplot as plt import numpy as np y = np.array([35, 25, 25, 15]) fig, _ = plt.subplots() plt.pie(y) fig

Pie chart with legend and names:

import matplotlib.pyplot as plt import numpy as np y = np.array([35, 25, 25, 15]) mylabels = ["Apples", "Bananas", "Cherries", "Dates"] fig, _ = plt.subplots() plt.pie(y, labels = mylabels) plt.legend(title = "Four Fruits:") fig
