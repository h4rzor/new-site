Python is a purpose programming language just like Java, C# C, C++ and so on. It is created in 1991, so long time ago. Being general purpose it was widely used for web, back-end and fron-end development throughout the years, but in the recent years, the hype around Artificial Intelligence is at all high. And Python became the most widely used programming language for AI due to its simplicity(some code looks like pseudo code) and its readability. It hides much of the complexity of other lower level languages such as Java, C++. It works on different platforms, Linux, Windows and Mac. It became the go-to language for beginners in the recent years.
In this site I'm going to use it to showcase the methods and logic behind Machine Learning algorithms as well as Deep Learning ones. But first things first, let's begin by quick introduction to Python.
